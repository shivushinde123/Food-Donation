# Food Donation Management System

A premium web application built with Vue 3 and Vite for managing food donations.

## How to Run in VS Code

1. **Open the Project Folder**:
   Open VS Code and select `File > Open Folder...`, then navigate to the `Food Donation Management` directory.

2. **Open the Integrated Terminal**:
   Press ``Ctrl + ` `` (backtick) or go to `Terminal > New Terminal`.

3. **Install Dependencies**:
   If you are running this for the first time, run:
   ```bash
   npm install
   ```

4. **Start the Development Server**:
   Run the following command to start the app:
   ```bash
   npm run dev
   ```

5. **View the Application**:
   Once the server starts, it will provide a link (usually `http://localhost:5173`). Click it or copy it into your browser.

## Tech Stack
- **Frontend**: Vue 3 + Vite
- **Styling**: Vanilla CSS (Glassmorphism)
- **Icons/Emoji**: Integrated native emojis
